Merlin, an assistant for editing OCaml code

Merlin is an editor-independant tool to ease the developpement of programs in OCaml. It aims to provide features available in modern IDEs such as:

  • "As you type" error reporting
  • Auto completion
  • Extraction of type information
  • Source browsing ("jump to definition")

See the following video for a demonstration of these features:

Integration with emacs and vim is already present, and should come to Sublime Text soon.

Merlin is available from opam or directly from sources on github and information on how to set it up is available on the wiki.
